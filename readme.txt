1. После чекаута репозитория в корневой папке выполнить команду:
	pip install -r requirements.txt
   которая запустит процесс установки всех необходимых пакетов.
2. После завершения установки войти в папку django_root и запустить сервер командой:
	./manage.py runserver
3. Открыть в браузере http://127.0.0.1:8000/
